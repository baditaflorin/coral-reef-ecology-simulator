# Privacy

This v1 static site does not collect analytics, personal data, or telemetry.

User preferences and simulator snapshots are stored locally in the browser with `localStorage`. They are not transmitted to the project maintainer.

The app includes outbound links to:

- https://github.com/baditaflorin/coral-reef-ecology-simulator
