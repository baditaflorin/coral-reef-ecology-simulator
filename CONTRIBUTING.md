# Contributing

Thanks for considering a contribution.

## Local Workflow

1. Install Node.js 22 or newer.
2. Run `npm install`.
3. Run `make install-hooks`.
4. Use Conventional Commits, such as `feat: add reef controls`.
5. Run `make test` and `make smoke` before pushing.

## Project Principles

- Keep the public deployment static on GitHub Pages.
- Never commit secrets or `.env` files with real values.
- Prefer small, focused changes with tests for simulation logic.
