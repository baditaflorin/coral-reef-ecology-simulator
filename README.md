# Coral Reef Ecology Simulator

Live site: https://baditaflorin.github.io/coral-reef-ecology-simulator/

Repository: https://github.com/baditaflorin/coral-reef-ecology-simulator

An interactive reef simulator where species, climate stress, visuals, and sound reveal trophic cascade dynamics.

This repository is being scaffolded as a static GitHub Pages project. The simulator runs entirely in the browser.
